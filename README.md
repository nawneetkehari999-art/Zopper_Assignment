# Zopper Business Intelligence Internship Assignment

This project simulates an insurance business portfolio and performs analytical evaluation using Python and data visualization.

## Project Objective
The goal of this assignment is to analyze policy sales and claim data to understand business performance, claim trends, and profitability.

## Tools & Technologies
- Python (Pandas, Matplotlib)
- Jupyter Notebook
- Data Visualization

## Dataset
The dataset contains simulated insurance policy and claims data including:

- Customer_ID
- Vehicle_ID
- Policy_Tenure
- Premium
- Claim_Amount
- Claim_Date
- Policy_Purchase_Date

## Analysis Performed
The following analyses were conducted:

1. Total premium collected in 2024
2. Monthly claim cost analysis for 2025–2026
3. Claim to Premium ratio by policy tenure
4. Month-wise claim ratio
5. Future claim liability estimation
6. Premium earned calculation

## Visualization
A monthly claim trend visualization was created using Matplotlib to understand claim patterns over time.
